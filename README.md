# nonsense Digital Synthesizer
nonsense is a digital synthesizer VST created for use in music production in a Digital Audio Workstation such as Ableton Live. It was created using the JUCE audio framework (https://juce.com/). It also has a standalone build.

## Build Info
This project uses Visual Studio's build tools. Install Visual Studio and open the .sln file (in Builds/VisualStudio2022/nonsense.sln, JUCE structures projects strangely for some reason), then click Build > Build Solution at the top of your screen.
